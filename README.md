☕ Code Coffee - Gestão Inteligente de Escalas
🏆 1º Lugar na Jornada de Inovação SEBRAE

Uma plataforma SaaS inovadora desenvolvida para automatizar e otimizar a gestão de escalas de trabalho em pequenas e médias empresas (PMEs), com foco inicial no setor de saúde.

📌 Visão Geral do Projeto
O Code Coffee nasceu para resolver o "caos administrativo" de empresas que operam em regime de turnos 24/7. A criação manual de escalas é lenta, gera conflitos de horários e riscos jurídicos por descumprimento das normas da CLT.

Nossa solução transforma esse processo através de um algoritmo de validação em tempo real, garantindo conformidade legal e eficiência operacional.

🛠️ Stack Tecnológica
Core: React 18 (Hooks, Context API)

Build Tool: Vite (Para performance extrema em desenvolvimento)

Estilização: [CSS3 / Tailwind CSS] (Ajuste conforme o que usou)

Qualidade de Código: ESLint & Prettier

Versionamento: Git (GitFlow)


✨ Funcionalidades Principais
📅 Gerador de Escalas Automático: Criação de turnos baseada na carga horária e disponibilidade.

⚖️ Validador de Compliance (CLT): Alertas em tempo real sobre violações de descanso obrigatório (11h interjornada) e carga horária semanal.

📊 Dashboard de Gestão: Visão macro para o administrador sobre a cobertura de turnos e custos.

📱 Interface Responsiva: Desenvolvida com foco no trabalhador que acessa a escala via mobile e no gestor que opera via desktop.

🔄 Gestão de Trocas: Fluxo de solicitação e aprovação de trocas de plantão centralizado.


🏗️ Arquitetura e Boas Práticas
O projeto foi construído seguindo rigorosos padrões de engenharia de software para garantir escalabilidade:

Componentização: UI modularizada para reaproveitamento e facilidade de teste.

Clean Code: Nomenclatura semântica de variáveis e funções, seguindo o guia de estilo do React.

Princípio de Responsabilidade Única (SRP): Separação clara entre componentes de interface e lógica de negócios.

Performance: Uso de Vite para garantir o Hot Module Replacement (HMR) mais rápido do mercado.

Bash

npm install

# 4. Inicie o servidor de desenvolvimento
